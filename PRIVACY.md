# 개인정보 처리방침

Click Faster는 사용자의 개인정보를 수집, 저장, 판매, 공유하지 않습니다.

## 수집하지 않는 정보

- 방문한 웹사이트 주소
- 동영상 시청 기록
- 계정 정보
- 결제 정보
- 위치 정보
- 입력한 텍스트나 페이지 콘텐츠

## 저장하는 설정

확장은 브라우저의 확장 저장소에 다음 설정만 저장합니다.

- 확장 사용 여부
- 빠른 재생 배속
- 작동 방식: 누르는 동안 또는 클릭 전환
- 속도 표시 여부
- 사이트 호환 모드 여부
- 사용자가 직접 제외한 사이트의 도메인

이 설정은 확장 기능을 유지하기 위한 값이며 외부 서버로 전송하지 않습니다.

## 권한 사용 목적

- `storage`: 사용자가 선택한 배속과 작동 방식을 저장합니다.
- `activeTab`: 팝업에서 현재 사이트 도메인을 확인해 해당 사이트 제외 설정을 저장합니다.
- `<all_urls>`: 사용자가 방문하는 여러 사이트의 HTML 동영상을 대상으로 배속 기능을 적용합니다.

## 외부 전송

Click Faster는 서버를 운영하지 않으며, 확장 내부에서 외부 분석 도구나 광고 SDK를 사용하지 않습니다.
